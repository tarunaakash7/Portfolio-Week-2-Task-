## 🎨 Customization Guide

### Change Name
```html
<span class="name-text">YOUR NAME HERE</span>
```

### Change Colors
```css
:root {
  --pu: #8b5cf6;  /* Change this */
  --cy: #06b6d4;  /* And this */
  /* etc */
}
```

### Update Projects
Find the JavaScript section and modify the `projs` array:
```javascript
const projs = [
  {
    num: '01',
    t: 'Your Project Title',
    d: 'Your description',
    tags: [{ t: 'Technology', c: 'pu' }],
    ac: 'pu'
  }
];
```

### Update Skills
Modify the `skillCats` array with your skills

### Update Contact
Find the contact section and update your email/links
