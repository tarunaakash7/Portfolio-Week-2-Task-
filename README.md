## 🎨 Customization Examples

### Change Your Name
Open `final-working-portfolio.html` in a text editor:
```html
<span class="name-text">YOUR NAME</span>
```

### Change Colors
Find the `:root` section:
```css
:root {
  --pu: #8b5cf6;    /* Change purple */
  --cy: #06b6d4;    /* Change cyan */
  /* etc */
}
```

### Update Contact Email
```html
<a href="mailto:your-email@gmail.com" class="cl">
  your-email@gmail.com
  <span class="cb">Email ↗</span>
</a>
```

## 💡 Tips & Tricks

### Test on Different Devices
1. Open portfolio in browser
2. Press F12 (Open DevTools)
3. Click device icon (top-left of DevTools)
4. Select different devices
5. See responsive design in action

### Edit the File Safely
1. Make a backup copy first
2. Open with text editor (VS Code, Notepad, etc)
3. Edit content or colors
4. Save file (Ctrl+S)
5. Refresh browser (F5)

### Troubleshooting
- **Fonts not loading?** → Check internet connection, fonts have fallbacks
- **Colors look different?** → Clear browser cache (Ctrl+Shift+Delete)
- **Not responsive?** → Check DevTools zoom (Ctrl+0)
